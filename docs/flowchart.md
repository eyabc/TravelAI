```mermaid
  A[앱 실행] --> B{로그인 여부 확인}
  B -- Yes --> C[메인 화면 진입]
  B -- No --> D[로그인/회원가입 화면]
  D --> E[회원가입 or 로그인 수행] --> C

  C --> F{기존 큐레이션 있음?}
  F -- Yes --> G[최근 큐레이션 이어서 시작]
  F -- No --> H[새 큐레이션 시작 버튼 노출]

  H --> I[마법사 안내 메시지]
  I --> J[프롬프트 기반 질문 응답 흐름]
  J --> K[응답 기반 일정/명소 큐레이션 생성]

  G --> K

  K --> L[추천 일정 화면]
  L --> M[마이 리스트 추가]
  L --> N[명소 상세 보기 + 번역]
  L --> O[일정 공유]

  C --> P[네비게이션 메뉴 열기]
  P --> Q[프로필 설정 화면]
  P --> R[프롬프트 저장/불러오기 화면]
  P --> S[환경정보 조회 / 현재 위치 기반 추천]
  P --> T[오프라인 캐시 화면 접근]
  P --> U[긴급 알림 수신 화면]

```
